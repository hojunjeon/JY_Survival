# 수정
1. 무기 사용 딜레이가 너무 김
    - 버그는 최대 3대 때려야 죽는데, 플레이어는 1초에 1번 공격
    - 버그 처치까지 시간이 너무 오래 걸림
2. 버그가 플레이어를 때리는, 반대로 내가 버그를 때리는 효과가 미미함
    - 맞았는지, 때렸는지 확인이 어려움
3. 버그의 공격 딜레이가 너무 짧음
    - 컨트롤 미스하면 hp가 한번에 쭉 닳아버림
4. 사망 시 게임이 계속 진행됨
    - hp가 0이 되어 사망해도 사망 메세지나 게임 오버 화면이 뜨지 않고 게임이 계속 진행됨
5. 배경
    - 검은색 밖에 없음
    - 화면에 보이는 맵이 너무 좁음
    - 플레이어가 맵 바깥으로 이동 시 화면이 따라 오지 않음
