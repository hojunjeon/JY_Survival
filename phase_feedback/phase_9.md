## #수정

- [ ] **공격 방향 개선 — 자동 에임**: 현재 C/C++ 무기는 이동 방향으로만 공격이 나가, 적을 등지고 도망치면서 공격(카이팅)이 불가능하다. 모든 투사체 무기의 공격 방향을 **가장 가까운 적을 자동 타겟팅**하는 방식으로 변경. 타겟이 없을 경우 마지막 이동 방향 유지.

- [ ] **Input Mismatch 스폰 타이밍 수정**: 게임 시작 직후 10~15초 만에 컨트롤 반전 이펙트가 발동돼 신규 플레이어가 이탈한다. Input Mismatch 및 Infinite Loop 스폰을 **웨이브 8 이상(약 90초 이후)**으로 제한. WaveSystem.js 웨이브 테이블에 타입별 최소 등장 웨이브 조건 추가.

- [ ] **적 접촉 시 넉백 추가**: 현재 적이 플레이어에 닿으면 자석처럼 달라붙어 체력이 연속으로 빠진다. 플레이어가 적에게 접촉 데미지를 받을 때 **플레이어를 반대 방향으로 60~80px 튕겨내는 넉백** 효과 추가. 무적 시간(0.5초) 중에는 추가 넉백 없음.

- [ ] **'REVERSED CONTROLS' 텍스트 위치 수정**: 현재 조작 반전 경고 텍스트가 캐릭터 위치를 따라다니며 주변 적을 가린다. HUD 텍스트를 **화면 상단 고정 위치(y: 60px)**로 이동. 캐릭터 추종 방식 제거.

- [ ] **Python 무기 밸런스 상향**: 현재 투사체 속도와 연사력이 낮아 초반 웨이브 돌파가 버겁다. 투사체 속도 +20%, 쿨타임 10% 단축.

## #추가

- [ ] **웨이브 타입별 최소 등장 조건 추가**: WaveSystem.js에 타입별 `minWave` 필드 도입. 기준안:
  - Memory Leak: 웨이브 1+
  - Race Condition: 웨이브 5+
  - Infinite Loop: 웨이브 8+
  - Input Mismatch: 웨이브 8+
  - Library Dependency: 웨이브 10+ (보스 직전)
