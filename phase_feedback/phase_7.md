# 수정 및 보완 사항

## #수정

- [ ] **재시작 시 잔상 버그**: 게임 오버 후 R키로 재시작(startGame) 시 이전 플레이어 이미지가 화면에 잔류함. ctx.clearRect 타이밍 또는 엔티티 배열 초기화 누락 의심. 렌더링 완전 초기화 보장 필요.
- [ ] **피격 이펙트 바운딩 박스 노출**: 몬스터 피격 시 스프라이트보다 훨씬 큰 흰색 사각형이 깜빡임. 보스 2페이즈 폭주 시에도 붉은 반투명 사각형 바운딩 박스가 그대로 노출됨. 스프라이트 외곽선에 맞춘 클리핑/알파 처리 필요.
- [ ] **이벤트 화면 텍스트 상단 쏠림**: 이벤트 대화창 박스에서 텍스트가 상단에 붙어 있음. 상하 중앙 정렬 깨짐. 박스 내 상하 패딩/수직 중앙 정렬 적용 필요.
- [ ] **이벤트 화면 동기 일러스트 비율/크기**: 이벤트 등장 시 캐릭터 전신이 화면을 과도하게 가림. 얼굴·상반신 컷인(Cut-in) 형식으로 크롭해서 보여주도록 수정.
