# 수정 및 보완
1. 맵 중간에서 버그가 생성되는 오류
    - 갑자기 앞에 생성되서 피하기 어려움

2. 이벤트 표시 화면의 대화창 사이즈가 작음
    - 텍스트가 대화창 안에 다 들어가지 못함

3. E3 클리어 시 무기 추가 안됨( 보스전도 확인 필요 )

4. 사망 시 게임 오버 화면에서 게임 시작 버튼이 안보임
    - R키를 눌러 재시작 시 사망 시점의 캐릭터와 몬스터의 잔상이 배경에 남음


# 추가 및 개선
1. 이벤트 화면 구성 변경
    - ex. 아래 예시를 디벨롭 할 것
    ```
    [ <E1> 이벤트 : 들여쓰기 지옥 ]
    -----------------------------------------------
    박진우 
    -----------------------------------------------
    아니 이거 또 이러네? 
    지윤님 저 진짜 들여쓰기 오류 때문에 미치겠습니다..
    -----------------------------------------------
    박진우가 들여쓰기 오류로 인해 컴퓨터를 향해 주먹을 쥡니다.
    그를 진정시키기 위해 IndentationError를 처치하세요 (0/15)
    ```
    - 두번째 이벤트 화면도 유사한 방식으로 구성하고 피드백 요청 할 것

2. 장선형 보스 개선안
    - 공격 패턴이 너무 단순함(추가적인 패턴이 있으면 좋을 듯)
    - 보스가 정사각형이라 위엄이 안삼(이미지 교체 필요) -> 김지윤과 동일하게 픽셀 이미지로(크기는 더 크게)

3. 기본 버그들의 체력 조정
    - 낮은 체력 : 2대 / 중간 체력 : 3대 / 높은 체력 4대
    - MVP 이후 스테이지가 올라갈 때 마다 체력은 증가할 수 있도록

4. 이벤트 클리어 조건에 맞는 양의 이벤트 몬스터만 소환
    - E1 : 15마리
    - E3 : 3마리
    - 기본 몹들은 이벤트 발생 시 즉시 사라지며 이벤트 클리어와 동시에 다시 등장하기 시작
    - 이벤트 몹들은 특별한 공격이 탑재되어야 함(예: 플레이어를 향해 돌진, 플레이어를 향해 총알 발사 등)

5. 이벤트 발생과 보스 등장 화면에서 NPC의 대사와 함께 픽셀 이미지를 띄워주기
    - 우선은 각각의 랜덤 인물 픽셀이미지를 생성해 사용
    - 추후 개인의 실물사진 이미지 파일을 주면 그에 맞는 픽셀 이미지를 생성하고 교체

6. 몬스터들이 피격될 때 효과 추가
    - 몬스터가 피격될 때마다 몬스터의 색깔이 변하며 피격되었음을 알림
    - 몬스터가 죽을 때도 동일한 효과를 적용
