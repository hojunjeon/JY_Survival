## 아이템 수집

- 자동 흡수: 플레이어 반경 60px 이내 아이템 자동 획득
- 흡수 연출: 아이템 → 플레이어 방향으로 이동 후 소멸 (0.3s)
- 인벤토리 한도: 12슬롯 (강화 재화·HP 포션은 수량 누적, 나머지는 개별 슬롯)

---

## 아이템 효과

| 아이템 | 효과 | 소모 방식 |
|--------|------|----------|
| 강화 재화 | 무기 레벨업 재료 | 레벨업 시 자동 소모 |
| HP 포션 | HP +20 즉시 | 획득 즉시 적용 |
| 공속 버프 | 공격속도 +30% / 10초 | 획득 즉시 적용 |
| 방어막 | 다음 피격 1회 무효 | 획득 즉시 적용 |
| 무기 파편 | 진화 재료 (Cycle 4 연동) | 진화 시 소모 |
| 전설 코어 | 무기 레벨 즉시 MAX | 획득 즉시 적용 |
| 환경 해제권 | 스테이지 기믹 디버프 제거 | 획득 즉시 적용 |

## 빌드 검증형 스테이지 기믹

`stages/StageConfig.js`에 스테이지별 상수 하드코딩:

```js
export const STAGE_WEAK_WEAPONS = {
  2: ['python', 'javascript'],   // Stage 2: 공속 ×2배 감소
  3: ['javascript', 'linuxbash'] // Stage 3: 데미지 ×0.5
};
```

- 디버프 적용: 스테이지 진입 시 해당 무기 보유 시 자동 적용
- 디버프 표시: HUD 무기 슬롯에 빨간 경고 아이콘
- 해제: 환경 해제권 아이템 획득 시 제거
