## Cycle 1 — 수정

### 1. 사망 처리
- **문제**: `player.isDead = true`가 되어도 게임 루프가 계속 진행됨
- **해결**: `game.update`에서 `player.isDead` 감지 → 게임 오버 화면 표시 + 재시작(새로고침 또는 메뉴 복귀)

### 2. 이벤트 UI 연결
- **문제**: `EventSystem`이 이벤트를 트리거하지만 `main.js`에 `EventModal` 연결 없음 → 화면에 아무것도 안 뜸
- **해결**: `event_triggered` 알림 수신 시 게임 일시정지 + 이벤트 모달 표시. 조건 달성 시 보상 화면 후 재개.

### 3. 피격 효과 (Hit Feedback)
- **문제**: 플레이어가 맞거나 적을 때려도 시각적 피드백이 없어 타격감이 없음
- **해결**:
  - 적 피격 시: 적 스프라이트 0.1초 흰색 플래시
  - 플레이어 피격 시: 화면 가장자리 0.15초 붉은 비네팅

### 4. 적 접촉 데미지 쿨다운
- **문제**: 플레이어·적 충돌 감지가 매 프레임 실행 → 순간적으로 HP가 다 닳음
- **해결**: 플레이어에 `contactDamageCooldown` 타이머 추가. 피격 후 0.5초 무적.

### 5. 배경 및 카메라
- **문제**: 배경이 검은색, 맵이 좁아 보임, 플레이어가 화면 밖으로 나가도 카메라가 따라오지 않음
- **해결**:
  - 월드 좌표계 도입: 플레이어 중심 카메라 (ctx translate)
  - 배경: 그리드 패턴 또는 타일 배경으로 공간감 부여
  - 맵 경계: 플레이어 이동 범위를 월드 크기로 제한

### 6. 이벤트 발생 타이밍
- **문제**: 플레이 중 이벤트가 발생하지 않는 것처럼 보임
- **해결**: 위 이벤트 UI 연결(#2)로 동시 해결. 별도 로직 변경 없음.
  - 현재 트리거 타이밍: E1=30초, E3=90초 (유지)

## Cycle 2 — 추가

### 1. 타이머 HUD 표시
- **현황**: `HUD.js`에 타이머 렌더 로직 구현 완료. `main.js`에 HUD 인스턴스 미연결.
- **해결**: `main.js` 렌더 루프에 `HUD.render()` 호출 추가. 기존 인라인 HP 텍스트 대체.

### 2. 버그 픽셀 아이콘
- **현황**: 버그 6종 모두 단색 도형으로 표시 중
- **해결**: `PixelRenderer.js`에 버그별 32×32 픽셀 아트 추가
  - SyntaxError: 빨간 느낌표 형태
  - NullPointerException: 반투명 유령 형태
  - SegFault: 깨진 조각 형태
  - 회복 버그: 녹색 하트 형태
  - IndentationError (이벤트): 들여쓰기 오류 형태
  - EnvError (이벤트): 경고 삼각형 형태
