# Phase 8 Cycle 2 추가 — 신규 몬스터 5종 계획

> 작성일: 2026-04-18

## 아키텍처 결정

Enemy.js의 `ENEMY_STATS` + `update()` 분기 방식으로 신규 타입 추가.
스프라이트 없으면 폴백 색상 사각형으로 렌더 (나중에 교체 가능).
main.js에서 특수 로직 처리가 필요한 타입은 update 루프에 분기 추가.

| 몬스터 | Enemy.js 변경 | main.js 변경 |
|--------|-------------|-------------|
| Race Condition | `race_condition` 타입 추가, `linkedEnemy` 참조 | 동시 처치 판정 (1초 내) |
| Memory Leak | `memory_leak` 타입, `growTimer`, `scale` 필드 | 가비지 오브젝트 배열 관리 |
| Infinite Loop | `infinite_loop` 타입, 원형 공전 AI | 코드 벽 오브젝트 관리 |
| Input Mismatch | `input_mismatch` 타입, 디버프 투사체 발사 | Input.js reverseControls 플래그 |
| Library Dependency | `library_dependency` 타입 | 주변 적 방어력 버프 (takeDamage 수식) |

## 태스크

- T1: Race Condition (Enemy.js + main.js)
- T2: Memory Leak (Enemy.js + main.js)
- T3: Infinite Loop (Enemy.js + main.js)
- T4: Input Mismatch (Enemy.js + Input.js + main.js)
- T5: Library Dependency (Enemy.js + main.js)
