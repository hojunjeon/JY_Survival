# Phase 3 Cycle 2 — 추가 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 타이머 HUD를 main.js에 연결하고, 버그 6종에 16×16 픽셀 아트 스프라이트를 추가하여 단색 사각형을 대체한다.

**Architecture:** HUD.js는 이미 완전히 구현되어 있으나 main.js에서 인라인으로 HP 텍스트만 렌더링 중이다. EventSystem의 elapsed/totalKills/상태값을 HUD.render()에 연결하여 타이머·킬카운트·이벤트 상태를 표시한다. 버그 스프라이트는 PixelRenderer.js에 BUG_SPRITES 맵으로 추가하고, Enemy.render()에서 스프라이트가 있으면 drawSprite()를 사용, 없으면 기존 색상 사각형으로 폴백한다. 스프라이트는 16×16(scale=2 렌더 → 화면 32×32)으로 설계한다.

**Tech Stack:** Vanilla JS, HTML5 Canvas, Vitest

render(ctx) {
  const sprite = PixelRenderer.BUG_SPRITES[this.type];
  if (sprite) {
    // 16×16 스프라이트를 scale=2로 → 화면 32×32
    PixelRenderer.drawSprite(ctx, sprite, this.x - this.width / 2, this.y - this.height / 2, 2);
    return;
  }

  // 폴백: 단색 사각형 (알 수 없는 타입)
  const colors = {
    syntax_error:      '#ff4444',
    null_pointer:      'rgba(180,180,255,0.7)',
    seg_fault:         '#884400',
    heal_bug:          '#44ff88',
    indentation_error: '#ffaa44',
    env_error:         '#4488ff',
    enemy:             '#ff8800',
  };
  ctx.fillStyle = colors[this.type] || colors.enemy;
  ctx.fillRect(this.x - this.width / 2, this.y - this.height / 2, this.width, this.height);
}
