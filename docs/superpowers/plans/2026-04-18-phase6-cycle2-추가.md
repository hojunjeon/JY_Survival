## T7: Screen Shake — 모든 적 피격 시

**파일**: `main.js` (또는 `core/Camera.js` 있으면 거기에)

**구현 방법**:
- `screenShake = { intensity: 0, duration: 0 }` 상태 추가
- 적 피격 이벤트 발생 시 `triggerScreenShake(intensity=3, duration=0.1)` 호출
- `render()` 최상단에서 `ctx.save()` → `ctx.translate(randX, randY)` → 전체 렌더 → `ctx.restore()`
  - `randX = (Math.random()-0.5)*2 * intensity`
  - `intensity`는 duration 남은 비율에 따라 감쇠
- 보스 피격 시: intensity=6, duration=0.15
