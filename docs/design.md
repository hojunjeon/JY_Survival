> 최종 업데이트: 2026-04-14 | Phase 1 브레인스토밍 완료

---

## 기술 스택

- **런타임**: 브라우저 (로컬 서버 불필요)
- **구현**: Vanilla JS + HTML5 Canvas, ES6 모듈
- **스프라이트**: Canvas API로 직접 그리는 32×32 픽셀 아트
- **개발 서버**: `npx serve .` (ES6 모듈 CORS 이슈 방지)

## 핵심 게임 루프

```
무기 선택 → 버그 웨이브 생존
  → [시간 경과] 이벤트1 발생 (일시정지 + 동기 대사 알림)
    → 조건 달성 → 강화 재화 획득 → 재개
  → [시간 경과] 이벤트2 발생 (일시정지 + 동기 대사 알림)
    → 조건 달성 → 무기 획득 → 재개
  → [이벤트2 클리어 + 추가 시간 경과] 장선형 등장
    → 보스 처치 → 스테이지 클리어 → 무기 + 강화 재화 획득
퀘스트 Q1: 버그 100마리 처치 (이벤트 버그 포함) → 스탯 업그레이드 선택
```

## 스테이지 구조 (모든 스테이지 공통)

| 순서 | 내용 | 보상 |
|------|------|------|
| 퀘스트 Q1 | 버그 100마리 처치 (스테이지 전반 누적) | 스탯 업그레이드 선택 (이동속도/공격속도/피해량) |
| 이벤트 1 | 시간 기반 트리거, 이벤트 버그만 등장 | 강화 재화 |
| 이벤트 2 | 시간 기반 트리거, 이벤트 버그만 등장 | 무기 획득 |
| 보스 장선형 | 이벤트2 클리어 + 추가 시간 후 등장 | 무기 + 강화 재화 |

- 이벤트 발생 시: 게임 일시정지 + 동기 대사 + 이벤트 알림 화면
- 이벤트 클리어 시: 게임 일시정지 + 보상 선택 화면 → 선택 후 재개

## 무기 시스템

| 무기 | 공격 방식 | 획득 방법 |
|------|---------|---------|
| Python | 360° 자동 투사체 (범위 넓음, 느림) | 시작 선택 |
| C/C++ | 전방 고속 단일 관통탄 | 시작 선택 |
| Java | 주변 오비탈 오브젝트 소환, 자동 공격 | 시작 선택 |
| Git | 범위 스턴/롤백 CC기 | 이벤트/보스 보상 |
| SQL | 광역 쿼리 공격 (쿨타임 김) | 이벤트/보스 보상 |
| JavaScript | 랜덤 방향 투사체 난사 | 이벤트/보스 보상 |
| Django | 전방 부채꼴 범위 공격 | 이벤트/보스 보상 |
| Linux/Bash | 화면 전체 궁극기 (쿨타임 매우 김) | 이벤트/보스 보상 |

**무기 강화**: 강화 재화 소모, 100% 성공률, 단계별 비용 증가  
**무기 보유 한도**: 스테이지당 최대 4개  
**스테이지 진입 시 무기 선택**:
- Stage 1: Python / C/C++ / Java 중 1개 선택
- Stage N (N≥2): 소유 무기 중 N개 선택 (최대 4개)

## 적 (버그) 시스템

| 버그 | 이동 | 체력 | 접촉 데미지 | 특이사항 |
|------|------|------|------------|---------|
| SyntaxError | 직선, 보통 속도 | 보통 | 보통 | 기본형, 다수 등장, 빨간 느낌표 외형 |
| NullPointerException | 빠름 | 낮음 | 낮음 | 히트앤런, 반투명 유령 외형 |
| SegFault | 느림 | 높음 | 높음 | 탱커형, 깨진 조각 외형 |
| 회복 버그 | 빠름 (도망) | 낮음 | 없음 | 처치 시 HP 아이템 드롭 |
| EnvError (이벤트 전용) | 느림 | 높음 | 보통 | 미처치 시 이동속도 감소 장판 생성 |

> 디버프 시스템 (피격 시 이동속도/공격속도 감소 등)은 MVP 이후 추가 예정.
